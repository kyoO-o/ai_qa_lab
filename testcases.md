# Login Feature – Test Cases (Updated with Edge Cases)

## Functional, Negative, Boundary, Security, Edge Test Cases

| Test Case ID | Title                                | Steps                                                                                                        | Expected Result                                                               |
| ------------ | ------------------------------------ | ------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------- |
| FT-01        | Correct login                        | 1. Бүртгэлтэй имэйл оруулах<br>2. Зөв нууц үг оруулах<br>3. Login товч дарах                                 | Dashboard руу амжилттай нэвтэрнэ                                              |
| FT-02        | Password visibility toggle           | 1. Нууц үг оруулах<br>2. Show/Hide toggle товч дарах                                                         | Нууц үг харагдах / нуух төлөв зөв солигдоно                                   |
| NT-01        | Incorrect password – first attempt   | 1. Зөв имэйл оруулах<br>2. Буруу нууц үг оруулах<br>3. Login товч дарах                                      | Login амжилтгүй, "Incorrect password" мессеж гарна                            |
| NT-02        | Incorrect password – second attempt  | 1. Зөв имэйл оруулах<br>2. Буруу нууц үг дахин оруулах<br>3. Login товч дарах                                | Login амжилтгүй, "Incorrect password" мессеж гарна                            |
| NT-03        | Account lock after 3 failed attempts | 1. Зөв имэйл оруулах<br>2. Буруу нууц үг 3 удаа дараалан оруулах                                             | "Account temporarily locked" мессеж гарна                                     |
| NT-04        | Empty email validation               | 1. Имэйл талбарыг хоосон үлдээх<br>2. Нууц үг оруулах<br>3. Login товч дарах                                 | Validation error: "Email is required"                                         |
| NT-05        | Empty password validation            | 1. Имэйл оруулах<br>2. Нууц үг хоосон үлдээх<br>3. Login товч дарах                                          | Validation error: "Password is required"                                      |
| BT-01        | Maximum email length                 | 1. 254 тэмдэгттэй имэйл оруулах<br>2. Зөв нууц үг оруулах<br>3. Login товч дарах                             | Систем зөв ажиллана эсвэл validation error гарна                              |
| BT-02        | Minimum password length              | 1. Хамгийн бага урттай нууц үг оруулах<br>2. Зөв имэйл оруулах<br>3. Login товч дарах                        | Login амжилттай эсвэл "Password too short" алдаа гарна                        |
| BT-03        | Email with special characters        | 1. Имэйлд `+`, `.`, `-` гэх мэт тэмдэгтүүдтэй имэйл оруулах<br>2. Зөв нууц үг оруулах<br>3. Login товч дарах | Login амжилттай эсвэл зөв Validation error гарна                              |
| BT-04        | Leading/trailing spaces in email     | 1. Имэйлд зайгаар эхлэх/дуусгах<br>2. Зөв нууц үг оруулах<br>3. Login товч дарах                             | Систем зайг автоматаар тайлж, login зөв ажиллана                              |
| BT-05        | Leading/trailing spaces in password  | 1. Нууц үгт зайгаар эхлэх/дуусгах<br>2. Имэйл оруулах<br>3. Login товч дарах                                 | Систем зайг автоматаар тайлж, login зөв ажиллана эсвэл validation error гарна |
| ST-01        | SQL Injection attempt                | 1. Имэйлд `admin' OR '1'='1` оруулах<br>2. Нууц үг оруулах<br>3. Login товч дарах                            | Login амжилтгүй, SQL Injection хамгаалалт ажиллана                            |
| ST-02        | XSS attack attempt                   | 1. Имэйлд `<script>alert(1)</script>` оруулах<br>2. Нууц үг оруулах<br>3. Login товч дарах                   | Script ажиллахгүй, input sanitize хийгдсэн байна                              |
| ST-03        | Brute force attack protection        | 1. Буруу нууц үгийг олон удаа хурдан оруулах                                                                 | 3 дахь оролдлогын дараа account түгжигдэнэ                                    |
| EC-01        | Email in uppercase                   | 1. Имэйл бүх том үсгээр оруулах<br>2. Зөв нууц үг оруулах<br>3. Login товч дарах                             | Login амжилттай (имэйл case-insensitive)                                      |
| EC-02        | Password with unicode characters     | 1. Нууц үгт Unicode тэмдэгтүүд оруулах<br>2. Бүртгэлтэй имэйл оруулах<br>3. Login товч дарах                 | Login зөв ажиллана эсвэл validation error                                     |
| EC-03        | Rapid login attempts                 | 1. Login товчийг хурдан дарах, хэд хэдэн оролдлого хийх                                                      | System lockout эсвэл throttle mechanism ажиллана                              |
